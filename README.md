# E-commerce Admin Panel
## Project Overview
This project is a backend-focused admin panel where only authenticated admins and admin user can access and manage the system. It helps streamline ecommerce operations like product management, coupon management, category management, user management.
## Features
• Developed a role-based admin panel to manage users, products, categories, and orders  
• Implemented authentication and authorization for secure admin access  
• Built complete CRUD operations for core modules (products, categories, users)  
• Integrated order management and coupon functionality  
• Implemented email-based password reset system for secure account recovery  
• Designed responsive UI using Bootstrap 5 Template 
• Used MySQL for database management with optimized queries             
• Followed Service-Repository architecture for clean and maintainable code 
## Tech Stack
- PHP(Laravel 12)
- MySQL
- Blade Template 
- Bootstrap 5
- jQuery add ajax
